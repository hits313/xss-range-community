# XSS Range — Community

Public hub for the [**XSS Range**](https://xss-range.vercel.app) training platform by [@hits313](https://github.com/hits313).

The training-range source lives in a private repo. This is the public surface for:

| | |
|---|---|
| 🚩 **Anti-cheat disputes** | If your account was flagged and you believe it was a mistake, [open an issue](https://github.com/hits313/xss-range-community/issues/new?template=anti-cheat-dispute.md&title=%5BDispute%5D+account+%3Chandle%3E) with your handle, the labs you solved, and a short write-up of HOW you solved each. |
| 💡 **Lab suggestions** | New lab ideas, vuln classes you'd like covered, real-world surfaces you've seen — [open a feature request](https://github.com/hits313/xss-range-community/issues/new?template=lab-suggestion.md&title=%5BSuggestion%5D+). |
| 🐛 **Bug reports** | Platform broken? Lab unsolvable? Solve registers as cheat when it shouldn't? [Open a bug](https://github.com/hits313/xss-range-community/issues/new?template=bug-report.md&title=%5BBug%5D+). |
| ✍️ **Write-ups** | Cleared a lab? Drop your write-up [here](https://github.com/hits313/xss-range-community/discussions) (Discussions). |
| ☕ **Support** | https://buymeacoffee.com/shahhits12u |

## Code of conduct

- Disputes are handled in good faith. Don't flame, don't threaten, don't dox.
- Submitted write-ups should be sanitised — no working payloads against third-party targets, no PII, no secrets.
- Lab-source itself is private; please don't ask for the source.

— @hits313
